yass

yass stands for Yet Another Scheduling Simulator. It is written in C and is released under the GPL-2 license. It can be downloaded here. Several uniprocessor and multiprocessor scheduling algorithms are implemented, for example RUN and U-EDF. To build all the scheduling algorithms based on LPDPM, you'll need to download cplex and move it into the schedulers/cplex/ directory.
